# Ebbinghaus Memory - 功能文档索引

## 项目简介

Ebbinghaus Memory 是一款基于艾宾浩斯遗忘曲线与 SM-2（SuperMemo-2）算法的间隔重复记忆卡片应用。通过科学的复习调度，帮助用户高效记忆知识。支持传统被动复习（闪卡翻转）与主动输出练习（填空、连词、翻译、造句）相结合的学习模式。

**技术栈**：Next.js 15 · React 19 · TypeScript · Prisma 7 · PostgreSQL · better-auth · Tailwind CSS 4
